# Personal Portfolio Website

## Project Description
This project is a responsive personal portfolio website created using HTML, CSS, and JavaScript.  
It showcases personal information, skills, projects, resume, and contact details.

The website is designed to work on different devices such as mobile phones, tablets, and desktops.

---

## Features
- Responsive design
- Navigation menu
- About section
- Skills section
- Projects section
- Resume download option
- Contact section
- Basic JavaScript interaction

---

## Technologies Used
- HTML
- CSS
- JavaScript

---

## Folder Structure

portfolio/

│── index.html

│── style.css

│── script.js

│── resume.pdf

---

## How to Run the Project

1. Download or clone the repository
2. Open the project folder
3. Run `index.html` in any web browser
